# Contribution Graph

Превью доступно по следующему URL: https://xxphantom.github.io/contribution-graph/

### Установка

```bash
git clone https://github.com/xxphantom/contribution-graph.git
cd contribution-graph
npm install
```

### Запуск

```bash
npm run dev
```

### Сборка

```bash
npm run build
```

## Деплой на GitHub Pages

```bash
npm run predeploy
npm run deploy
```
